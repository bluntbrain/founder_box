# founder_box
Weight based vending proof of concept.

Hardware used -
  1. Raspberry pi 3
  2. RC522 RFID module
  3. Weighing machine (rs232 serial usb output)
  4. 12V soleniod door lock
  5. Buzzer
  6. Jumper wires
  
  Dependencies - 
  https://pypi.org/project/pad4pi/1.0.0/
